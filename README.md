## 📸 Features

- ➕ Add tasks with a single click or press **Enter**
- ✅ Mark tasks as complete with a checkbox
- 🗑️ Delete individual tasks
- 🔍 Filter by **All / Active / Done**
- 🧹 Clear all completed tasks at once
- 💾 **Auto-saves** tasks to `tasks.json` — persists after closing
- 📅 Timestamps each task with the date added
- 🌑 Sleek dark UI with purple accent colors

## 🚀 Installation

```bash
pip install customtkinter
```

## ▶️ Run

```bash
python todo_app.py
```

## 📁 Project Structure

```
todo_app/
├── todo_app.py     # Main application
├── tasks.json      # Auto-created on first run (task storage)
└── README.md
```

## 🧱 Tech Stack

| Library | Purpose |
|---|---|
| `customtkinter` | Modern UI widgets |
| `json` | Task persistence |
| `datetime` | Timestamps |

## 💡 OOP Design

- `ToDoApp` — main window, holds all state and logic
- `TaskCard` — reusable widget for each task row
